# Markdown
-облегченный язык разметки, созданный для форматирования текста с помощью простых символов, понятных человеку и легко конвертируемых в HTML-код.

## Основные возможности синтаксиса
Заголовки: `# Заголовок 1`, `## Заголовок 2` и т.д.

*Курсив*: `*курсив*`

**Жирный текст**: `**жирный текст**`

~~Зачеркнутый~~:  `~~зачеркнутый~~`

***Полужирный***:  `***полужирный***`

**Маркированный текст**:

- Пункт 1
  -  Вложенный пункт
- Пункт 2

 ```
-  Пункт 1
  -  Вложенный пункт
-  Пункт 2
 ```

1.   Пункт 1
2.   Пункт 2

 ```
1.  Пункт 1
2.  Пункт 2
 ```

[Ссылки](https://github.com/djitzy/Project_Unicorn_):  `[Ссылки](https://github.com/djitzy/Project_Unicorn_)`

Список задач:

- [x] Выполненная задача
- [ ] Невыполненная задача

 ```
- [x] Выполненная задача
- [ ] Невыполненная задача
 ```

Горизонтальный линии

---
***

```
---
***
 ```
